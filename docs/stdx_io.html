stdx_io.h

home

STDX - IO (file I/O utility functions) Part of the STDX General Purpose C Library by marciovmf License: MIT https://github.com/marciovmf/stdx

Overview

Provides utility functions for reading and writing files.

How to compile

To compile the implementation define X_IMPL_IO in one source file before including this header.

To customize how this module allocates memory, define X_IO_ALLOC / X_IO_FREE before including.
