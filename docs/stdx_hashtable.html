stdx_hashtable.h

home

STDX - Generic Hashtable Part of the STDX General Purpose C Library by marciovmf License: MIT https://github.com/marciovmf/stdx

Overview

Provides a generic, type-agnostic hashtable implementation with
customizable hash and equality functions. Supports arbitrary key
and value types, optional custom allocators, and built-in iteration. Includes helpers for common cases like string keys.

How to compile

To compile the implementation define X_IMPL_HASHTABLE in one source file before including this header.

To customize how this module allocates memory, define X_HASHTABLE_ALLOC / X_HASHTABLE_REALLOC / X_HASHTABLE_FREE before including.

Dependencies:

x_hashtable_create

Create a hashtable with key/value type inference using compile-time type name strings.

#define x_hashtable_create(tk, tv) x_hashtable_create_(\sizeof(tk), strcmp(X_TYPE_NAME(tk), "char*")==0, strchr(X_TYPE_NAME(tk), '*')!=0, \sizeof(tv), strcmp(X_TYPE_NAME(tv), "char*")==0, strchr(X_TYPE_NAME(tv), '*')!=0)

Parameters

tk
Key type (passed as a type token, e.g. int, XSmallstr, char*).
tv
Value type (passed as a type token, e.g. Foo, Bar*, char*).

Returns

Pointer to a newly created hashtable, or NULL on failure.

x_hashtable_create_full

Create a fully-configurable hashtable with custom hashing, comparison, cloning, and destruction.

XHashtable * x_hashtable_create_full(
  size_t key_size,
  size_t value_size,
  XHashFnHash fn_key_hash,
  XHashFnCompare fn_key_compare,
  XHashFnClone fn_key_copy,
  XHashFnDestroy fn_key_free,
  XHashFnClone fn_value_copy,
  XHashFnDestroy fn_value_free);

Parameters

key_size
Size in bytes of the key type.
value_size
Size in bytes of the value type.
fn_key_hash
Function used to hash keys.
fn_key_compare
Function used to compare keys for equality.
fn_key_copy
Function used to clone/copy keys into the table.
fn_key_free
Function used to destroy/free keys owned by the table.
fn_value_copy
Function used to clone/copy values into the table.
fn_value_free
Function used to destroy/free values owned by the table.

Returns

Pointer to a newly created hashtable, or NULL on failure.
