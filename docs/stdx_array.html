stdx_array.h

home

STDX - Array Part of the STDX General Purpose C Library by marciovmf License: MIT https://github.com/marciovmf/stdx

Overview

Provides a generic, dynamic array implementation with support for random access, insertion, deletion, and stack-like operations
(push/pop). Useful for managing homogeneous collections in C with automatic resizing.

How to compile

To compile the implementation define X_IMPL_ARRAY in one source file before including this header.

To customize how this module allocates memory, define X_ARRAY_ALLOC / X_ARRAY_REALLOC / X_ARRAY_FREE before including.
