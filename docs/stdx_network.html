stdx_network.h

home

STDX - Cross-Platform Networking API Part of the STDX General Purpose C Library by marciovmf License: MIT https://github.com/marciovmf/stdx

Overview

  • Provides a high-level, cross-platform socket networking API.
  • On windows, this library links with iphlpapi.lib and ws2_32.lib

How to compile

To compile the implementation define X_IMPL_NET in one source file before including this header.

To customize how this module allocates memory, define X_NET_ALLOC / X_NET_FREE before including.
